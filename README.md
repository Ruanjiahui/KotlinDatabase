# KotlinDatabase

这个依赖包主要解决了 android 使用本地数据库的繁琐操作 

好处是可以直接使用对象进行数据操作，不需要再次使用对象转成ContentValues那么麻烦


调用这个库的的链接

    compile 'ruan.database.com:mydatabase:1.1'
